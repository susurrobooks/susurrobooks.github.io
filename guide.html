susurro  Guide

책이 독자에게 닿는 길은 하나가 아닙니다.

이미 종이책으로 완성된 이야기도, 아직 원고로 머물러 있는 이야기도 전자책으로 다시 태어날 수 있습니다. susurro는 당신의 책이 가장 자연스럽게 독자에게 다가가도록 돕습니다.

제작 과정  Process

종이책이 이미 있는 경우

InDesign 파일 등을 바탕으로 EPUB 변환을 진행합니다.

(기존 책의 디자인을 유지하면서, 전자책에 맞는 최적화를 거칩니다.)

 

원고만 준비된 경우

원고 편집에서부터 디자인, 전자책 제작까지 전 과정을 함께합니다.

(처음부터 끝까지 완성형 eBook을 만들 수 있습니다.)

프로세스  Process

구분 종이책 → EPUB 변환 원고만 → 전체 제작
준비물 PDF / InDesign 파일 / 최종 교정본 텍스트 원고 (docx, hwp, txt 등)
주요 단계 원고 접수 → 구조 분석 → 디자인 최적화 → 기기별 검수 → 납품 원고 검토 → 교정·편집 → 디자인 → EPUB 제작 → 검수·납품
작업 범위 변환 중심 / 최적화 편집·디자인 포함 풀 패키지
디자인 기존 서식 단순화·최적화 표지 + 본문 레이아웃 디자인
비용 기준 분량, 서식 복잡도, 이미지·표·각주 원고 분량, 교정 범위, 디자인 난이도, 이미지·표 유무
추천 종이책 출간 후 전자책 제작 원고만 준비되어 전자책 제작 희망

비용 기준  Pricing

전자책 제작 비용은 작업 난이도와 페이지 수, 이미지 포함 여부 등에 따라 결정됩니다.

  • 기본 편집 및 EPUB 변환: 페이지 수 기준
  • 디자인 커스터마이징: 표지/내지 디자인 난이도에 따라 추가 비용 발생
  • 추가 기능(목차, 하이퍼링크, 인터랙티브 요소 등): 옵션에 따라 비용 산정

susurro와 함께라면 원고가 있든, 이미 책이 있든, 가장 자연스럽고 매끄럽게 전자책으로 완성할 수 있습니다. 제작 과정과 비용 기준을 이해하고 나면, 이제 당신의 책이 독자에게 닿을 준비가 됩니다.
